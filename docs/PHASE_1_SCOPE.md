# Phase 1 Scope

Phase 1 is local-only and focused on validating the core salary and client payment workflow without a backend.

Included scope:
- Manual employee management
- Manual salary history
- Manual salary outgoing tracking
- Manual client/payment tracking
- Reports
- Backup/restore
- CSV export
- Local PIN convenience lock

Phase 1 data is stored in browser localStorage. Use JSON backup before clearing browser data.
