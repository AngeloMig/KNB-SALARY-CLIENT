# KNB Salary & Client Payment Dashboard

A localStorage-based admin dashboard for managing employee salaries, salary outgoing/payroll runs, client payments, reports, and cash flow.

## Tech Stack

- HTML
- CSS
- jQuery
- Chart.js
- localStorage

## How To Run

Open `index.html` in a browser.

## Current Phase

Phase 1.2 localStorage MVP

## Important Limitations

- Data is saved only in the browser.
- Not multi-user yet.
- Local PIN is only a convenience lock, not real authentication.
- Use JSON backup before clearing browser data.
- Backend/database will come in Phase 2.

## Main Sections

- Dashboard
- Employees
- Salary History
- Salary Outgoing (5th and 20th salary releases)
- Clients
- Payments
- Reports
- Settings
